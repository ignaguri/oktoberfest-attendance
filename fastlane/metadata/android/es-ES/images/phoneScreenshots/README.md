# Capturas de pantalla — es-ES

Coloca las PNG finales en este directorio. Los archivos se leen en orden lexicográfico, así que el prefijo numérico controla el orden en Play Console.

## Especificaciones

- Formato: PNG (preferido) o JPEG
- Relación de aspecto: 9:16 vertical (u 16:9 horizontal)
- Tamaño: 1080×1920 px (vertical) es el objetivo seguro — Play Console admite hasta 7680 px en el lado largo
- Mín.: 2 capturas, Máx.: 8

## Convención de nombres

Idéntica a en-US, pero con la UI en español:

```
01_home.png
02_log_attendance.png
03_tents.png
04_group_leaderboard.png
05_profile.png
06_wrapped.png
07_map.png
08_friends.png
```

## Captura

Igual que en-US, pero antes cambia el idioma del dispositivo a español (Ajustes → Idioma y región) y dentro de la app (Perfil → Idioma).

## Superposiciones de texto

El feedback de los testers recomienda superposiciones cortas (p. ej. "¡Lleva la cuenta de tus cervezas!"). Mantén la tipografía y el tono coherentes en todas las capturas.
